# Иконки приложения

Поместите файлы иконок в эту папку:

## Требуемые форматы:

- **icon.png** - основная иконка (512x512 или 1024x1024 пикселей)
  - Используется для всех платформ
  - Рекомендуемый размер: 512x512 или 1024x1024

## Дополнительные форматы (опционально):

Для лучшей поддержки разных платформ можно добавить:

- **icon.icns** - для macOS (можно создать из PNG)
- **icon.ico** - для Windows (можно создать из PNG)

## Как создать иконки:

### Из PNG в ICNS (macOS):
```bash
# Используя iconutil (только macOS)
iconutil -c icns icon.iconset
```

### Из PNG в ICO (Windows):
Можно использовать онлайн-конвертеры или инструменты вроде ImageMagick.

## Текущая конфигурация:

Приложение ищет иконку по пути: `assets/icon.png`

Если файл не найден, будет использована иконка по умолчанию Electron.
